# Sistema de Pedidos Conibase

Sistema de registro de pedidos de piso e revestimento desenvolvido em Python.

## Funcionalidades

- Registrar pedido com nome do cliente, produto e quantidade
- Numero de pedido gerado automaticamente
- Listar todos os pedidos
- Buscar pedido por nome do cliente
- Dados salvos em arquivo JSON — nao perde ao fechar o programa

## Tecnologias

- Python 3
- JSON para persistencia de dados

## Como executar

```bash
python pedidos.py
```
