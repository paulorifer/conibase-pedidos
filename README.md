# Projetos Conibase — Python

Dois sistemas reais desenvolvidos em Python para a Conibase, 
empresa de materiais de construção.

## Sistema de Estoque

Funcionalidades:
- Cadastrar produto com nome, quantidade e preço
- Listar todos os produtos
- Atualizar quantidade em estoque
- Remover produto
- Dados salvos durante a sessão

##  Sistema de Pedidos

Funcionalidades:
- Registrar pedido com nome do cliente, produto e quantidade
- Número de pedido gerado automaticamente
- Listar todos os pedidos
- Buscar pedido por nome do cliente
- Dados salvos em arquivo JSON — não perde ao fechar o programa

## Tecnologias

- Python 3
- JSON para persistência de dados

## Como executar
python estoque.py
python pedidos.py
